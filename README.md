## Description

`py2exe` is a simple, minimalistic batch script that allows you to convert Python scripts (`.py`) into standalone executable (`.exe`) files using PyInstaller. It provides a clean user interface in the Windows command line and supports optional settings.

---

## Features

- Convert any Python `.py` file to a standalone `.exe`.
- Optionally assign an `.ico` icon.
- Optionally remove the console window from the output executable.
- Simple, user-friendly command line interface.

---

## Requirements

- Windows 10/11
- Python 3.x installed and added to PATH
- pip installed
- PyInstaller (the script will prompt to install it if missing)

---

## Usage

1. Download `py2exe.bat`.
2. Run the batch file (`double-click` or `cmd`).
3. Choose `1` to start conversion or `2` to exit.
4. If PyInstaller is not installed, follow the prompt to install it.
5. Enter the Python file path. (File you want to convert)
6. Optionally enter an icon (`.ico`) or leave blank.
7. Choose whether to remove the console window.
8. Confirm to start the conversion.
9. After conversion, the output `.exe` will be in the `dist` folder.
